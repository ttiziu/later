# Later

Aplicación minimalista de gestión de tareas construida con Next.js y TypeScript.

![Next.js](https://img.shields.io/badge/Next.js-16.0-black?logo=next.js&logoColor=white)
![React](https://img.shields.io/badge/React-19-61DAFB?logo=react&logoColor=white)
![TypeScript](https://img.shields.io/badge/TypeScript-5.8-3178C6?logo=typescript&logoColor=white)
![Tailwind CSS](https://img.shields.io/badge/Tailwind-4.1-38B2AC?logo=tailwind-css&logoColor=white)

## Características

→ Múltiples listas de tareas  
→ Completado de tareas con animación de tachado  
→ Edición inline de listas  
→ Mover tareas entre listas  
→ Reordenar tareas dentro de listas  
→ Tema oscuro/claro  
→ Persistencia en LocalStorage  
→ Diseño responsive

## Stack Tecnológico

**Framework** → Next.js 16  
**Lenguaje** → TypeScript  
**Estilos** → Tailwind CSS  
**Animaciones** → Motion  
**Componentes UI** → Radix UI, Base UI  
**Temas** → next-themes
